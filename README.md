# 여기로 모여라
이동 경로를 함께 공유하세요!

우리 앱은 친구들과 목적지로 가는 여정을 실시간으로 공유할 수 있도록 도와줘요.

방을 생성하고 친구들에게 코드만 공유하면, 모두가 같은 지도에서 서로의 현재 위치와 목표지점까지의 거리를 확인할 수 있어요.

지도에 표시된 마커와 바텀 시트를 통해 누가 가장 빠르게 목적지에 도착할지 등수도 확인할 수 있어요.

함께하는 이동이 더 즐거워질 거예요!

지금 바로 친구들과 함께 경로를 공유해보세요.

## 기술 스택
- riverpod: 상태관리 라이브러리로는 Riverpod을 사용했습니다. 앱에서 여러 화면 간 데이터 흐름을 효과적으로 관리하기 위해 선택했습니다. 이 라이브러리는 경량이면서도 강력한 상태 관리 도구로, 사용자의 위치 정보나 경로 상태를 실시간으로 반영하는 데 적합합니다.
- dio / retrofit: 서버와의 통신을 위해 강력하고 유연한 네트워크 라이브러리인 Dio를 사용했고, Retrofit을 통해 API 호출을 보다 효율적으로 관리했습니다. 이를 통해 위치 데이터나 기타 정보를 서버와 안전하게 주고받을 수 있었습니다.
- web_socket_channel: WebSocketChannel 객체를 생성하여 간단하게 소켓 연결을 구현할 수 있어 사용하고 있습니다.
- go_router: 기본 제공하는 Navigator 보다 패턴을 간결하고 선언적으로 처리할 수 있어 사용하고 있습니다.
